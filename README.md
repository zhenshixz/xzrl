# 📅 小智日历 (长辈关怀极简大字版日历)

专为长辈和视障群体设计的极简大字版日历工具。摒弃了繁琐多余的广告和复杂操作，专注提供最清晰、直观的时间与本地天气服务。

---

## 🛠️ 技术栈

*   **架构**：纯前端轻量级单页应用（SPA），无多余依赖，加载极速。
*   **语言**：`HTML5` + `Vanilla CSS` + `原生 JavaScript`
*   **字体系统**：接入 `Google Fonts (Noto Sans SC)`，支持极高清晰度的文字无损缩放。

---

## 🌟 核心功能

*   🗓️ **双历合一清晰展示**：超大字号清晰展示阳历（新历）、农历（旧历）、星期、二十四节气，并完整支持传统天干地支与生肖年份（如 `丙午年 四月初六`）。
*   ⚙️ **人性化长辈关怀设计**：
    *   **大字/特大字无级切换**：支持大字（`1.2x`）与特大字（`1.4x`）一键切换，保障长辈无需老花镜也能清晰阅读。
    *   **双色护眼主题**：提供「暖心红」与「天海蓝」两种高对比度的极简皮肤，有效缓解眼部疲劳。
*   📆 **直观的万年历月视图**：采用新历、农历双行并排显示的日历网格，方便长辈快速切换查看历史或未来日期。
*   🌦️ **实时气象看板**：自动定位并呈现今天与明天的精准气象数据，包含实时温度区间、天气状态趣味 Emoji、湿度与风力等，可点击随时刷新。
